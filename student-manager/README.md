# Student Manager

Plugin WordPress quan ly sinh vien bang Custom Post Type, meta box va shortcode.

## Tinh nang

- Custom Post Type `Sinh vien`
- Meta box: `MSSV`, `Lop/Chuyen nganh`, `Ngay sinh`
- Luu du lieu an toan voi nonce va sanitize
- Shortcode `[danh_sach_sinh_vien]` hien thi bang danh sach sinh vien

## Su dung

1. Kich hoat plugin trong WordPress.
2. Tao du lieu trong menu `Sinh vien`.
3. Chen shortcode sau vao bai viet hoac trang:

```text
[danh_sach_sinh_vien]
```
